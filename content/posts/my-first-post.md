## 星舰建造记录

### 为什么选择 Hugo

在众多星际造船技术中，我选择了 **Hugo** 来建造这艘星舰。原因如下：

1. **光速引擎** - Hugo 用 Go 语言编写，构建速度极快，就像装备了曲率引擎
2. **单体式结构** - 只需要一个可执行文件，结构简洁，维护方便
3. **丰富的蓝图库** - 有大量现成的主题可以选择和改造
4. **Markdown 协议** - 使用 Markdown 编写内容，简单直观，兼容性好

### 星舰设计理念

这艘星舰的设计灵感来自《三体》中的 **自然选择号** 战舰：

- **深空配色** - 深空蓝、引擎青、星光白
- **推进器光效** - 模拟战舰引擎的蓝色光芒
- **控制台界面** - 类似战舰控制面板的 UI 设计
- **星空背景** - 动态的星空和超光速特效
